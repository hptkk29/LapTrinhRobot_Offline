📊 Kết Nối Google Sheets
Lưu thông tin đăng ký của phụ huynh vào Google Sheets tự động qua Google Apps Script.
ℹ️ Cách hoạt động: Phụ huynh điền form → Web gọi Apps Script → Apps Script lưu vào Sheet. Không cần server, không cần database — hoàn toàn miễn phí.
1
Mở Google Sheet của bạn
Mở link Google Sheet đã cung cấp: 📊 Mở Google Sheet →
Đảm bảo bạn đã đăng nhập bằng tài khoản Google có quyền chỉnh sửa sheet này.
2
Mở Apps Script Editor
Trong Google Sheet, click vào menu: Extensions (Tiện ích mở rộng) → Apps Script
Một cửa sổ mới sẽ mở ra. Xóa toàn bộ code mặc định trong editor.
⚠️ Nếu không thấy menu "Extensions", Sheet có thể đang ở chế độ xem. Nhấn nút "Open in Sheets" hoặc "Edit" để vào chế độ chỉnh sửa.
3
Dán code Apps Script
Copy toàn bộ đoạn code dưới đây và dán vào Apps Script Editor:
// Sata Robo — Google Sheets Integration
// Sheet ID: 1sN0s_l71m9SrAm8hzdUmJEnsRM23FZuUJOFCbUSIAx0

function doPost(e) {
  try {
    // Mở Sheet đầu tiên
    var ss = SpreadsheetApp.openById(
      '1sN0s_l71m9SrAm8hzdUmJEnsRM23FZuUJOFCbUSIAx0'
    );
    var sheet = ss.getSheets()[0];

    // Tạo header nếu sheet trống
    if (sheet.getLastRow() === 0) {
      sheet.appendRow([
        'Thời gian', 'Họ và tên', 'Điện thoại',
        'Email', 'Cơ sở', 'Khóa học'
      ]);
      // Format header row
      sheet.getRange(1, 1, 1, 6).setFontWeight('bold')
        .setBackground('#6B21A8').setFontColor('#ffffff');
    }

    // Parse data từ request
    var data = JSON.parse(e.postData.contents);

    // Thêm row mới
    sheet.appendRow([
      new Date().toLocaleString('vi-VN', {timeZone:'Asia/Ho_Chi_Minh'}),
      data.name    || '',
      data.phone   || '',
      data.email   || '',
      data.center  || '',
      data.course  || ''
    ]);

    // Trả về success
    return ContentService
      .createTextOutput(JSON.stringify({status:'ok'}))
      .setMimeType(ContentService.MimeType.JSON);

  } catch(err) {
    return ContentService
      .createTextOutput(JSON.stringify({status:'error', msg: err.toString()}))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

// Test function — chạy thử trong editor

4
Chạy test thử (khuyến nghị)
Trước khi deploy, test thử bằng cách:
1. Chọn function testDoPost từ dropdown phía trên
2. Nhấn nút ▶ Run
3. Lần đầu chạy sẽ hỏi cấp quyền — chọn "Allow"
4. Kiểm tra Google Sheet xem có thêm dòng dữ liệu không
✅ Nếu dòng dữ liệu xuất hiện trong Sheet → Apps Script hoạt động đúng. Tiếp tục bước 5.
5
Deploy Apps Script thành Web App
1. Nhấn nút Deploy (góc phải trên) → New deployment
2. Nhấn biểu tượng ⚙️ (cài đặt) → chọn "Web app"
3. Điền thông tin:
TrườngGiá trị cần chọn
DescriptionSata Robo Landing Page v2
Execute asMe (tài khoản Google của bạn)
Who has accessAnyone ← Bắt buộc phải chọn này!
🚨 Bắt buộc chọn "Anyone" cho "Who has access". Nếu chọn sai, landing page sẽ không gửi được dữ liệu vào Sheet.
5. Nhấn Deploy
6. Copy Web app URL — dạng: https://script.google.com/macros/s/AKfycb.../exec
6
Cập nhật SCRIPT_URL vào HTML
Mở file sp2-offline_landing_v2.html bằng Notepad, VS Code hoặc bất kỳ text editor nào. Tìm dòng:
var SCRIPT_URL = 'YOUR_GOOGLE_APPS_SCRIPT_URL';
Thay bằng URL vừa copy:
var SCRIPT_URL = 'https://script.google.com/macros/s/AKfycb.../exec';
✅ Lưu file và reload trên browser. Form đăng ký giờ đây sẽ lưu dữ liệu vào Google Sheets!

📘 Facebook Pixel & Retargeting
Theo dõi khách truy cập để chạy quảng cáo bám đuổi (retargeting) trên Facebook/Instagram.
ℹ️ Pixel hoạt động thế nào? Pixel theo dõi ai đã vào web nhưng chưa đăng ký → cho phép bạn chạy quảng cáo hiển thị lại cho đúng những người đó trên Facebook/Instagram.
1
Tạo Facebook Pixel
1. Vào Facebook Business Manager
2. Menu trái → Events Manager
3. Nhấn + Connect Data Sources → Web → Next
4. Chọn Facebook Pixel → đặt tên (vd: Sata Robo Landing) → Create Pixel
5. Chọn "Add code manually"
6. Copy đoạn code Base Code (dạng !function(f,b,e,v...)
2
Dán Pixel code vào HTML
Mở file HTML, tìm đoạn comment này trong thẻ <head>:
<!-- [FACEBOOK PIXEL - BẮT ĐẦU] -->
<!-- !function(f,b,e,v ...
fbq('init', 'THAY_BANG_PIXEL_ID_CUA_BAN'); -->
<!-- [FACEBOOK PIXEL - KẾT THÚC] -->
Bỏ dấu comment (xóa <!-- và -->) và thay THAY_BANG_PIXEL_ID_CUA_BAN bằng Pixel ID thật:
<!-- [FACEBOOK PIXEL - BẮT ĐẦU] -->
<script>
  !function(f,b,e,v,n,t,s){if(f.fbq)return;n=f.fbq=function(){
  n.callMethod?n.callMethod.apply(n,arguments):n.queue.push(arguments)};
  if(!f._fbq)f._fbq=n;n.push=n;n.loaded=!0;n.version='2.0';
  n.queue=[];t=b.createElement(e);t.async=!0;t.src=v;
  s=b.getElementsByTagName(e)[0];s.parentNode.insertBefore(t,s)}(window,
  document,'script','https://connect.facebook.net/en_US/fbevents.js');
  fbq('init', '123456789012345'); // ← Thay bằng Pixel ID thật
  fbq('track', 'PageView');
</script>
<!-- [FACEBOOK PIXEL - KẾT THÚC] -->

📈 Google Analytics 4 (GA4)
Phân tích hành vi người dùng, nguồn traffic, tỷ lệ chuyển đổi và hiệu quả từng kênh.
1
Tạo tài sản GA4
1. Vào Google Analytics
2. Nhấn Admin (bánh răng) → Create Property
3. Đặt tên: Sata Robo Landing SP2
4. Timezone: Vietnam (GMT+7)
5. Currency: Vietnamese Dong (VND)
6. Nhấn Next → chọn Web → Nhập URL trang web
7. Copy Measurement ID (dạng: G-XXXXXXXXXX)
2
Dán GA4 code vào HTML
Tìm đoạn comment trong <head>:
<!-- [GOOGLE ANALYTICS GA4 - BẮT ĐẦU] -->
<!-- <script async src="...G-THAY_BANG_MA_GA4"></script> -->
<!-- [GOOGLE ANALYTICS GA4 - KẾT THÚC] -->
Bỏ comment và thay Measurement ID thật:
<!-- [GOOGLE ANALYTICS GA4 - BẮT ĐẦU] -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'G-XXXXXXXXXX'); // ← Thay bằng Measurement ID thật
</script>
<!-- [GOOGLE ANALYTICS GA4 - KẾT THÚC] -->
3
Sự kiện GA4 đã tích hợp sẵn
Khi phụ huynh đăng ký thành công, sự kiện sau được gửi tự động:
gtag('event', 'generate_lead', {
  course_name: 'Tên khóa học đã chọn',
  center: 'Cơ sở đã chọn'
});
Để xem trong GA4: Reports → Engagement → Events → tìm generate_lead
4
Thiết lập Conversion trong GA4
1. Vào GA4 → Admin → Events
2. Tìm event generate_lead
3. Bật nút "Mark as conversion"
4. Sau 24-48h sẽ thấy trong Reports → Conversions
✅ Khi đã mark conversion, Google Ads sẽ tự tối ưu chiến dịch để tìm đúng người có khả năng đăng ký cao nhất.
5
Dashboard báo cáo khuyến nghị
Các báo cáo quan trọng cần theo dõi:
Báo cáoVị trí trong GA4Dùng để
Users by SourceReports → AcquisitionBiết traffic đến từ đâu (Facebook, Zalo, Google...)
Landing PageReports → Engagement → Landing pageTỷ lệ bounced rate
ConversionsReports → ConversionsSố người đăng ký mỗi ngày
Real-timeReports → Real-timeTheo dõi trực tiếp khi chạy ads
